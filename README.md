# Superstore Data Warehouse

## 📌 Project Overview
This project transforms the **Superstore CSV dataset** into a **normalized PostgreSQL database** and prepares the data for analytics and business intelligence.

The workflow follows an **ETL pipeline** to clean, structure, and load the data into a relational database.

---

## 🛠 Technologies Used
- Python (Pandas)
- PostgreSQL
- SQLAlchemy
- SQL
- Jupyter Notebook

---

## 🔄 Project Workflow
1. Database Creation
2. Data Modeling & Normalization
3. Data Cleaning
4. ETL Data Loading
5. SQL Transformations
6. Analytical Views Creation

---

- Average Profit per Customer

- `superstore.csv` — raw dataset

## 🎯 Objective
Build a structured **analytical database** from raw sales data to support **business analysis and BI dashboards**.
